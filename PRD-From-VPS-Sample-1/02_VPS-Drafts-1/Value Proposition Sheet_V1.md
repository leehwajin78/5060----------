## 📢 1. Executive Summary — 핵심 가치 명제

**직함에 갇혀 있던 경력을 시장이 구매 가능한 언어와 B2B 자산으로 '완벽히 번역(Translation)'해 주는, 실질적인 무기화 대행 서비스**를 팝니다.

> **[Value Proposition Statement — 3 Layer]**
>
> | 레이어 | 한 줄 정의 |

## 🎯 2. Customer Profile — 타깃 분석 및 Jobs / Pains / Gains

> 타깃이 실제로 지갑을 열어 구매하려는 **본질**을 정의합니다.
> 고객은 '디자인'을 사는 것이 아니라, **'체면 방어'**와 **'실제 제안 가능한 자산'**을 삽니다.

### 2-1. Before vs After

| 시점 | 고객의 상태 |
| :--- | :--- |
| **Before** | 경력과 지식은 풍부하나, 어떤 가치를 주는지 한 줄로 설명하지 못함. SNS·프로필·이력서가 파편화되어 B2B 시장에서 신뢰를 잃고 외면받는 상태. |
| **After** | 누가 봐도 명확한 '문제 해결사'로 포지셔닝 완료. 즉시 강연/자문 요청에 대응 가능한 **50장 강의안과 B2B 제안서 PDF**를 손에 쥔 상태. |

### 2-2. Customer Map (Jobs–Pains–Gains)

| 분석 층위 | 핵심 내용 요약 |
| :--- | :--- |
| **타깃 (Who)** | 대기업 임원·대표·명장 등 10~20년 이상 실무를 거쳤으나, 독립된 개인으로서 수익 창출형 포트폴리오가 전무한 **5060 전환기 전문가**. |
| **과업 (Jobs)** | 직함 없이 독립된 개인으로서, 내 경험을 **실제 C-Level이나 기업 HR이 돈을 내고 살 수 있는 구체적 '세일즈 자산(강의안/제안서)'**으로 구조화하고 발간하는 것. |
| **고통 (Pains)** | **① 언어 상실:** "나는 이런 일을 해왔다"는 아는데, "그래서 당신 기업에 어떤 가치를 준다(ROI)"로 변환 불가. <br>**② 자산 분산:** 노션·블로그·제안서가 따로 놀아 신뢰도 저하. <br>**③ 실행 장벽 & 체면:** 어린 직원/프리랜서에게 묻자니 체면이 상하고, 스스로 PPT를 켜기엔 디지털 리터러시가 부족해 실행을 미룸(완벽주의). |
| **효용 (Gains)** | **① 물리적 산출물:** 당장 영업에 쓸 수 있는 정체성 언어와 하이엔드급 제안서/강의안 원본 50장 확보. <br>**② 투자 배수(ROI):** 고문 자문(300만) 1건 + 특강(200만) 3건만 수주해도 **결제 대금(880만) 즉시 회수가 가능하다는 확신.** |

## 💰 4. 수익 구조 설계 및 성장 전략

> 본 사업은 추상적인 '자기계발 코칭'이 아닙니다.
> **"기업이 즉각 지갑을 열 수 있는 B2B 물리적 영업 무기(제안서/강의안)"를 VVIP 의전으로 100% 완전 대행(Done-for-you)해 주기에, 타깃의 높은 가격 수용성(High-Ticket) 확보가 가능한 캐시카우(Cash-Cow) 모델**입니다.

### 4-1. 왜 돈이 벌리는가 — 핵심 수익 논리 3가지

| # | 수익 논리 | 설명 |
| :---: | :--- | :--- |
| **1** | **지불 명분이 확실한 ROI 구조** | 고객이 880만 원(Option B)을 지불해도, B2B 제안서·강의안 50장으로 **고문 자문 1건(월 300만 원) + 특강 3건(건당 200만 원)만 잡으면 즉시 원금 회수.** |
| **2** | **'체면 방어'와 '시간'의 등가교환** | 수십만 원을 아끼려 스스로 PPT를 만드는 것은 타깃의 체면을 구기고 완벽주의 스트레스를 유발. VVIP 대접을 받으며 인터뷰 구술만으로 끝나는 편안함에 수백만 원을 지불. |
| **3** | **AI 백업 → 이익률 극대화** | 마스터 브리프 AI 생성기가 기획 정보 구조화 시간을 8시간 → 1시간 이내로 축소. **순이익률 46~60% 선 안정 방어.** |

### 4-2. 가격 구조 — 닻내림(Anchoring) 효과 기반 상품 라인업

> 고객이 가장 비싼 상품을 "합리적인 투자"로 느끼도록 설계한 가격 전략입니다.

| 단계 | 상품명 | 가격 | 역할 및 논리 |
| :---: | :--- | :--- | :--- |
| **미끼** | AI 브랜딩 수치 진단 (5~10문항) | 30,000원 → **한시적 무료** | 탤런트뱅크 등에서 안 팔리는 이유를 객관적으로 짚어 주는 리포트. 콜드 리드 DB 확보. |
| **앵커** | **Option A:** 뼈대 매니지먼트 패키지 | **6,500,000원** | 마스터 브리프 + 에셋 12종. 직접 영업·문서 편집이 가능한 소수 전문가 대상. Option B 대비 앵커 역할. |
| **⭐ 주력** | **Option B:** 100% 완전 대행 풀스택 | **8,800,000원** | Option A + **B2B 실전 제안서 + 시그니처 강의안 50장.** "230만 원만 더 내면 노동이 0이 되고, 즉시 강연료 회수 가능"이라는 결제 논리 무장. |
| **리텐션** | 에이전시 리테이너 (월정액 업데이트) | **월 50~100만 원** | 영업 시즌/트렌드에 따른 제안서 피보팅 + 알럼나이(Alumni) 네트워크 종속 → 고정 현금 흐름 창출. |

### 4-3. 성장 전략 로드맵 및 즉시 실행 과제

| 시점 | 집중 과제 | 채널·파트너십 활용 |
| :--- | :--- | :--- |
| **단기 (1~3개월)** | 지인·수강생 중 절박한 타깃 2~3명 핀셋 섭외 → 성공 포트폴리오(Before & After) 확보. | 원티드긱스·리멤버에서 프로필 부실 5060 식별 → AI 진단 체험 링크 다이렉트 송부. |

**🔥 Top 4 즉각 실행 액션:**

1. 마케팅용 노션 랜딩페이지(가설 페이지) **1일 내 제작** → 타깃 10명에게 발송.
2. 카카오톡/Typeform 등 No-code 기반 **5문항 AI 진단 프로토타입 2일 내 구현.**
3. 실 고객 데이터를 주입하여 **15p 브리프 렌더링 퀄리티** 정성 검증 (프롬프트 코어 확인).
4. 퀄리티 컨트롤을 위한 **A급 디자인 에이전시 시범 계약** 조율.

## 🚀 5. Job-Feature Map 기반 MVP 구현 상세 계획

> 위 §1~§4에서 확정한 비즈니스 논리가 **실제로 작동**하려면 반드시 선행되어야 할 기술 장치가 있습니다.
> 이 장에서는 타깃 고객의 과업(Jobs)이 솔루션의 기능(Feature)으로 치환되는 구조와 함께, **'AI 마스터 브리프 MVP 및 진단기'** 개발 계획을 상술합니다.

### 5-1. 왜 이 MVP를 만드는가 — 개발 목표 및 기대 효과

| 목표 | 상세 설명 | 기대 수치 |
| :--- | :--- | :--- |
| **운영 시간 압축 (Core Engine)** | 코칭 인터뷰(녹취/메모)를 입력하면, 프롬프트 처리로 15p 분량의 마스터 브리프 초안을 자동 생성. 대표 1인의 월 소화 물량을 극적으로 확장. | 건당 작업 시간 **48시간 → 30분 이내** / 월 2건 → **월 6건 이상** |
| **리드 확보 퍼널 (Lead Agent)** | 잠재 고객이 5~10문항 진단에 응답하면 B2B 진출 약점 리포트를 즉시 출력. VVIP 디렉팅 상담으로 전환 유도. | 리포트 완독자 대비 **CTA 전환율 10% 이상** |
| **무자본 쾌속 개발 (Speed)** | Vercel + Supabase(무료 티어) + Claude API를 결합한 1인 바이브코딩 환경에서 최단기 배포. | 프로토타입 **3주 내 런칭** |

### 5-2. 핵심 범위 정의 (In / Out / Next)

| 구분 | ✅ 포함 (MVP IN) | ❌ 제외 / 🔜 후속 (OUT · NEXT) |
| :--- | :--- | :--- |
| **Front-End (UI)** | 5~10문항 진단 폼 + 간단 대시보드 리포트(웹뷰) | ❌ 화려한 모션 디자인 · 🔜 V2에서 STT(Speech-to-Text) 연동 |
| **Back-End (Data)** | Next.js(Vercel) 구동, Supabase 리드 DB 적재 | ❌ 앱 형태 결제·SNS 회원가입 (이메일 수집으로 대체) |
| **Core AI (Biz-Logic)** | 응답 기반 1p 요약 브리프 + 관리자용 15p 상세 브리프 추출 (168개 프롬프트 패턴) | ❌ PPT 디자인 자동 Export (V2 이후) |

### 5-3. 아키텍처 개요 — 2-Track 사용자 흐름

```
                        ┌─────────────────────────────────────────────────────┐
  [TRACK 1: 리드 퍼널]   │  랜딩페이지 → 무료 진단 폼(5문항) → Claude API 분석  │
  (Front-Stage)          │  → 브랜드 지수·약점 리포트 출력 → VVIP 상담 CTA     │
                        └─────────────────────────────────────────────────────┘
                                              ↓ 전환
                        ┌─────────────────────────────────────────────────────┐
  [TRACK 2: 코어 엔진]   │  결제 완료 → 42문항 인터뷰 녹취/메모(Raw) 입력       │
  (Back-Stage Console)   │  → System Prompt 기반 Claude 모델 논리 변환          │
                        │  → [가치선언문 / 핵심 타깃 / 강의안 목차 / 제안서      │
                        │     뼈대 구조] 텍스트 일괄 반환 → 관리자 검수·발주     │
                        └─────────────────────────────────────────────────────┘
```

1. **TRACK 1 — 리드 제너레이션 (Front-Stage):**
   랜딩페이지 유입 → 무료 진단 폼 접속 → 5개 핵심 문항(객관식 90% + 단답형 10%) 응답 → Claude API 즉시 분석 → **"당신의 브랜드 지수 및 약점 분석 리포트"** 웹 출력 → VVIP 매니지먼트 상담 신청 CTA 클릭.

2. **TRACK 2 — 콘텐츠 엔진 렌더링 (Back-Stage Console):**
   결제·인터뷰 수집 완료 후, 운영자가 인터뷰 Raw 텍스트를 관리자 콘솔에 입력 → 자사 기획 철학이 이식된 System Prompt 기반 Claude 모델이 B2B 제안서 언어로 논리 변환 → **[가치선언문, 핵심 타깃, 1차 강의안 목차, 제안서 뼈대 구조]** 텍스트 일괄 반환 → 관리자 검수 후 디자인 외주 발주.
